## Status de conformidade

As [Diretrizes de Acessibilidade para Conteúdo Web (WCAG)](https://www.w3.org/WAI/standards-guidelines/wcag/) definem requisitos para que designers e desenvolvedores melhorem a acessibilidade para pessoas com deficiência. Elas definem três níveis de conformidade: Nível A, Nível AA e Nível AAA.

O **[Nome do Site/Produto]** está **parcialmente em conformidade** com o nível AA das WCAG 2.2. Conformidade parcial significa que algumas partes do conteúdo não estão totalmente em conformidade com o padrão de acessibilidade.

## Especificações técnicas

A acessibilidade de **brunopulis.dev.br** depende das seguintes tecnologias:

- HTML
- WAI-ARIA
- CSS
- JavaScript

Essas tecnologias são usadas em conjunto com navegadores modernos e tecnologias assistivas para proporcionar uma experiência acessível.

### Recursos de acessibilidade

**brunopulis.dev.br** inclui os seguintes recursos de acessibilidade:

- **Link de pular** para ignorar a navegação e ir direto ao conteúdo principal
- **Hierarquia lógica de títulos** para navegação com leitores de tela
- **Regiões de referência** (`<header>`, `<nav>`, `<main>`, `<footer>`) para navegação com tecnologias assistivas
- **Navegação completa por teclado** para todos os elementos interativos
- **Suporte às teclas de seta, Home e End** para componentes de abas
- **Tecla Escape** fecha modais e menus com gerenciamento de foco adequado
- **Contraste de cores** em conformidade com os requisitos WCAG AA
- **Temas claro e escuro** com suporte às preferências do usuário
- **Design responsivo** adaptando-se a diferentes tamanhos de tela
- **Indicadores de foco visíveis** para navegação por teclado
- **Suporte a movimento reduzido** respeitando as preferências do usuário
- **Atributos ARIA** em todos os componentes
- **Rótulos de leitor de tela** para elementos interativos
- **Atributos de idioma** para pronúncia correta
- **HTML semântico** para significado Estrutura
- **Validação de formulário** com identificação clara de erros e sugestões
- **Alternativas de texto** para conteúdo não textual
